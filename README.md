## Getting Started

1. Download the presentation

   ```sh
   git clone git@github.com:chadjvw/training-day-dynamodb.git
   ```

2. Install dependencies

   ```sh
   yarn install
   ```

3. Start the webpack server. The server will run at [`localhost:3000`](http://localhost:3000).

   ```sh
   yarn start
   ```
